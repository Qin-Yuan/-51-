### 介绍

分享一个之前完成的设计项目——基于51单片机智能鱼缸设计与实现；但是，它不是那么智能，也并没有投入到实际的鱼缸中做测试，这里仅作为一个综合性系统项目分享，养鱼需谨慎哦。该功能板具备以下功能：

- 温度实时监测功能；
- 按时投食鱼料功能；
- 外部时钟精确计时，掉电不会丢失时间；
- 外部存储电路，通过按键设置水温、投食时间、时钟校准、水循环周期、充氧周期等参数，并存储在外部，掉电仍保存数据；
- LCD1602显示鱼缸当前状态；
- 三个扩展继电器外接加热设备、充氧设备、水循环设备。
